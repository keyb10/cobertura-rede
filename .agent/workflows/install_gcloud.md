---
description: Install Google Cloud CLI
---

1.  **Download the Installer**:
    Click this link to download the installer: [Google Cloud SDK Installer](https://dl.google.com/dl/cloudsdk/channels/rapid/GoogleCloudSDKInstaller.exe)

2.  **Run the Installer**:
    -   Run the downloaded `.exe` file.
    -   Follow the on-screen prompts.
    -   **Important**: Make sure to check the boxes for "Start Google Cloud SDK Shell" and "Run 'gcloud init'" at the end of the installation.

3.  **Verify Installation**:
    After installation, open a **new** terminal window (or restart your current one) and run:
    ```powershell
    gcloud --version
    ```
